# quotes

**Purpose:** Fixes quotation mark inconsistencies between 1917 and FSB Bible versions.

**Files Changed:**
- `FSB.xml` - Corrected quotation marks to match 1917 edition pattern

**Workflow:** Compares quotation marks in FSB.xml against the 1917 version line-by-line. Detects and fixes: mismatched opening/closing quotes, incorrect quote types (" vs '), and quotes that shouldn't be present. Updates FSB.xml to align quotation style with the traditional 1917 edition.
